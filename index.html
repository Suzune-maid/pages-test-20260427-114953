Screen Wake Lock API 測試

主人，這頁是拿來測手機會不會自動鎖定。

如果瀏覽器支援 navigator.wakeLock，並且頁面保持前景，啟用後應可避免螢幕自動熄滅或自動鎖定。

頁面狀態 可見中

頁面載入後會檢查支援狀態。請用手機瀏覽器開啟，然後按下「啟用防鎖定測試」。

預期效果

  • 頁面保持前景時，螢幕不應因閒置而自動熄滅。
  • 手機不應因閒置而自動進入鎖定畫面。
  • 如果切到背景、切分頁或省電限制介入，Wake Lock 可能被釋放。

實測方式

  1. 用手機開這個頁面。
  2. 按「啟用防鎖定測試」。
  3. 把手機放著，超過平常自動鎖定時間。
  4. 如果螢幕仍亮著，代表效果符合預期。

限制與注意

  • 必須是 HTTPS 頁面，GitHub Pages 符合這個條件。
  • 通常需要使用者互動後才會成功請求。
  • 不同手機／瀏覽器支援度不同，iPhone Safari 常比 Android Chrome 更受限。
  • 手動按電源鍵或系統極端省電時，它不一定能阻止關閉螢幕。
